## 📌 Resumen

Internet nació como un proyecto militar (ARPANET) para sobrevivir ataques nucleares mediante una red descentralizada. Con la adopción de TCP/IP en 1983 se unificaron todas las redes, y en 1991 la World Wide Web lo convirtió en una herramienta masiva y pública.

---

## 🕒 Línea de tiempo

| Año | Hito |
|-----|------|
| 1969 | ARPANET — primeros 4 nodos conectados (UCLA, Stanford, UCSB, Utah) |
| 1972 | Invención del email |
| 1983 | Adopción de TCP/IP — protocolo universal entre redes |
| 1991 | World Wide Web — Tim Berners-Lee (CERN) |
| 1993 | Mosaic — primer navegador gráfico |
| 1995 | Internet comercial y acceso público masivo |
| 2000s | Web 2.0 — redes sociales y contenido de usuario |
| 2010s | Internet móvil, cloud, IoT |

## 🔑 Conceptos clave

| Concepto | Explicación simple |
|----------|--------------------|
| **Conmutación de paquetes** | Los datos se dividen en trozos (paquetes) que viajan por rutas distintas y se reensamblan al llegar |
| **ARPANET** | Primera red del Departamento de Defensa de EE.UU., creada para sobrevivir fallos en la red |
| **TCP/IP** | Protocolo común que permite que cualquier red se comunique con cualquier otra |
| **World Wide Web** | Capa de páginas enlazadas sobre Internet, inventada por Tim Berners-Lee en 1991 |
