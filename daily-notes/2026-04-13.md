## 🧠 Captura rápida (formato Cornell)

| 🔑 Claves / Preguntas            | 📝 Notas y desarrollo                                                                                    |
| -------------------------------- | -------------------------------------------------------------------------------------------------------- |
| ¿Qué es la dopamina?             | Neurotransmisor que regula la motivación, el aprendizaje, la atención selectiva y la toma de decisiones. |
| ¿De dónde se sintetiza?          | Se sintetiza a partir del aminoácido **L-Tirosina**.                                                     |
| ¿Por qué es clave para aprender? | El cuerpo la usa fundamentalmente para obtener aprendizaje; regula motivación y atención selectiva.      |
| ¿Cómo obtenerla por dieta?       | Carne, huevos, lácteos, legumbres y frutos secos.                                                        |

**📌 Resumen Cornell (2-3 frases propias):**

La dopamina es un neurotransmisor esencial para aprender: sin ella no hay motivación ni atención selectiva. Se fabrica a partir de **L-Tirosina**, un aminoácido que podemos obtener fácilmente con proteína animal o vegetal en la dieta.
