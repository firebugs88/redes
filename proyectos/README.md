# 🚀 Proyectos

Un `.md` por **proyecto activo**, creado con `templates/proyecto.md`. Una idea de `ideas/` se convierte en proyecto cuando madura (regla 2 de CLAUDE.md). Toda nota que pertenezca a un proyecto debe estar enlazada desde el `🔗 Relacionado` del proyecto.

> [!info] Proyecto ≠ idea
> Idea = semilla sin compromiso. Proyecto = tiene objetivo, próximos pasos y estado (`#estado/activo` · `#estado/pausado` · `#estado/completado`).
