# 🗄️ Archivo

Notas **degradadas o archivadas** por la skill `decantador-cognitivo`. Esto **no es una papelera**: nada se borra. Una nota llega aquí cuando su Índice de Valor es bajo (poca conectividad, baja relevancia, sin repaso, antigua) y tú aprobaste moverla.

> [!info] Reversible
> Mover una nota aquí **no rompe sus `[[enlaces]]`**: Obsidian los resuelve por nombre de archivo aunque esté en otra carpeta. Si una nota archivada vuelve a ser relevante, devuélvela a su subcarpeta de `investigaciones/` y regenera el índice con `indexador-grafo`.
